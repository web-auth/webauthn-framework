# Security Policy

## Supported Versions

| Version | Supported          |
|---------| ------------------ |
| 4.x.x   | :white_check_mark: |
| 3.3.x   | :white_check_mark: |
| < 3.3.x | :x:                |

## Reporting a Vulnerability

If you think you have found a security issue, DO NOT open an issue. You MUST email your issue: security AT
spomky-labs.com.
